HTML Events

HTML events are "things" that happen to HTML elements.

exmple of events :

  • An HTML button is clicked
  • A web page has finished loading
  • The mouse moves over an element
  • A keyboard key is pressed
  • An HTML input field is changed

often, when events happen u may want to do something, when javascript lets you execute code in response to events. HTML allows event handler attributes, with JavaScript code, to be added to HTML elements.
