## Overview

## Syllabus & Learned Topics

### 1. Fundamentals
- **Introduction**: Basic output, syntax, and linking scripts.
- **Variables**: Declaring variables using `var`, `let`, and `const`. Understanding block scope vs function scope.
- **Data Types**: String, Number, Boolean, Object, Array, Undefined. Using `typeof`.
- **Type Conversion**: Converting between strings, numbers, and booleans.

### 2. Operators & Logic
- **Arithmetic & Assignment**: Basic math and value assignment.
- **Comparison & Logical**: Understanding `>`, `<`, `==`, `===` and Logical `&&`, `||`.
- **Short-circuit Evaluation**: How `&&` and `||` handle non-boolean values.

### 3. Control Flow
- **Conditionals**: Making decisions with `if`, `else`, `switch`.
- **Loops**: Repeating code with `for` and `while` loops.
- **Flow Control**: Using `break` and `continue`.

### 4. Data Structures
- **Arrays**: Creating arrays, accessing elements.
    - **Methods**: `push`, `pop`, `shift`, `unshift`, `splice`, `slice`.
    - **Sorting & Searching**: `sort`, `reverse`, `find`, `filter`.
    - **Iteration**: Looping through arrays.
- **Objects**: specialized file structure.
- **Strings**: Manipulation, methods (slice, substring), and Template Literals.
- **Numbers**: Properties and methods.

### 5. Functions
- **Basics**: Declaration and invocation.
- **Advanced**: Arrow functions vs Traditional functions.

### 6. The DOM (Document Object Model)
- **DOM API**: Understanding the DOM tree.
- **Selectors**: Finding elements by ID, Class, Tag, QuerySelector.
- **Manipulation**:
    - **Content**: `innerHTML`, `textContent`.
    - **Styles**: `style` property, CSS classes.
    - **Animation**: Basic DOM animation.

### 7. Events
- **Event Listeners**: `addEventListener`, bubbling vs capturing.
- **Types**:
    - Mouse: `click`, `mouseover`, `mouseout`.
    - Keyboard: `keydown`, `keyup`.
    - Load: `onload`.
    - Timing: `setTimeout`, `setInterval`.

### 8. Web Forms
- **Validation**: validating user input.

### 9. Advanced Concepts
- **Scope**: Global, Function, and Block scope.
- **Hoisting**: Variable and function hoisting behaviors.
- **Strict Mode**: `"use strict"` directive.
- **Error Handling**: `try`, `catch`, `throw`, `finally`.
