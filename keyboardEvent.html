Keyboard Events happen when the user presses a key on the keyboard

  • keydown (key is pressed down)
  • keyup (key is released)
  • keypress (key is pressed down and held, deprecated, use keydown instead Still works for now, but you should NOT use it anymore because it will be removed or replaced in the future)

keypress only fires for character keys (a or 5), not for control keys(alst or backspace). Developers are advised to use keydown or keyup instead.
